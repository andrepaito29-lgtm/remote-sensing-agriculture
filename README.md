# Agricultura mediante Teledetección

Repositorio enfocado en el **monitoreo de cultivos mediante UAV (drones)**, análisis de **índices de vegetación** y **etiquetado de datos** para agricultura de precisión y monitoreo ambiental.

## Cultivos monitoreados
- Pitahaya
- Palma africana
- Pino

## Actividades principales
- Análisis de salud fisiológica del cultivo
- Evaluación de etapas fenológicas
- Clasificación de la vegetación y vigor vegetal
- Etiquetado geoespacial (polígonos, puntos y áreas)
- Integración de variables biológicas, edáficas y climáticas

## Herramientas y tecnologías
- UAVs (drones)
- QGIS
- Pix4D / WebODM
- Python
- Índices de vegetación (NDVI, SAVI, GNDVI)

## Enfoque
Aplicación de teledetección y análisis de datos para apoyar la **toma de decisiones agrícolas**, la **sostenibilidad** y el **monitoreo ambiental**.
